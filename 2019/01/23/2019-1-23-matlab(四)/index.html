摘要：MATLAB变量。

Matlab变量

MATLAB中的变量可以说是无类型的，但它的类型又只有数组或矩阵这两种，也就是说，MATLAB的变量只能是数组或者矩阵。

变量赋值

  • 简单的赋值

    通过上述赋值，输入去会返回一个结果，指明a的值为多少。但我们必须理解，即使这个变量只有一个元素，但它也是一个数组或者矩阵，是一个1*1的矩阵。

  • 表达式赋值

    我们可以通过表达式的方式赋值，如果只有一个表达式，没有变量，那么该表达式的值自动赋值给ans变量。

查看变量历史

可以通过who命令查看变量的历史，通过whos可以查看变量更加详细的信息，下面的命令也印证了变量的类型。

清除命令

通过clear可以从存储器清楚变量，如果后面不加变量名，默认清楚所有的变量。

多行命令

与python类似，如果需要用到多行的命令，可以用三个实心点扩展到另一行。

格式化输出

默认情况下MATLAB输出的格式是短格式，即数据是四位小数位数。但我们可以通过format来进行格式化输出。

命令 说明
format short 显示十进制后4位数字
format long 显示十进制后16位数字
format bank 将数字舍入到小数点后两位
format short e 以指数形式显示四位小数加上指数
format long e 以指数形式显示十六位小数加上指数
format rat 给出计算结果最接近的合理表达式
